# **Infinity autopilot**
Infinity è un autopilot avanzato totalmente open-source per UAV , inoltre permette compatibilità tra vari sistemi grazie ad un **Hardware Abstraction Layer** il quale permette attraverso vari environment diversi di essere compilato per vari sistemi e inoltre permette testing in **SITL**.


> [!WARNING]
> There is currently **no stable release** of **Infinity-Autopilot** available.  
> Il progetto è ancora in una fare **Iniziale** di sviluppo, le varie funzionalità saranno implementate nel prossimo periodo.


## Features

- EKF Error-state con delayed state
- Controller PID
- Controller LQR
- Controller MPC
- Logging avanzato
- Simulazione SITL
